# EbookReaderNX

### Usage :

- Move the .nro in /switch folder
- Move all ebooks in /switch/ebookReaderNX folder
- Warning : ebook title must not have '='
- Enjoy !

## Feature :
- Read book
- switch page by 10
- landscape/portrait
